# Tarefa

Crie a rota `POST /workout-plans/user/{idUser(uuid)}`
Utilizando o mesmo padrão da rota workout-plans, porém com o id do usuário por parametro.
Considere a mesma interface que está no banco de dados:

```ts
interface InputDto {
  userId: string;
  name: string;
  planGoal: string;
  workoutStartDate: DateTime?;
  workoutFinishDate: DateTime?;
  workoutDays: Array<{
    name: string;
    weekDay: WeekDay;
    isRest: boolean;
    estimatedDurationInSeconds: number;
    coverImageUrl?: string;
    workoutTime: string?; //responsavel pelo horario que a pessoa vai treinar. Ex: 15:00 ,14:30
    exercises: Array<{
      order: number;
      name: string;
      sets: number;
      reps: number;
      restTimeInSeconds: number;
      obs: string?;
      exerciseLoad: string?;
      exerciseId: string?
    }>;
  }>;
}

### Regras

- Só poderá criar o treino se o usuario da sessão for igual ao usuario passado no parametro OU se o usuario da sessão estiver cadastrado como personalId do usuario passado pelo parametro
- Se existir um treino já caadastrado e ativo, então ele desativar o anterior e cria esse
```
